# BottomShelfer

A customizable slide-up bottom sheet presentation controller for UIKit, available
pre-iOS 15 and supporting arbitrary height detents, a draggable grabber, an
optional dimming scrim, and full layout-metric overrides.

## Requirements

- iOS 13.0+
- Swift 6.0+

## Installation

Add the package via Swift Package Manager:

```swift
.package(url: "https://github.com/jonikay/BottomShelfer.git", from: "1.0.0")
```
